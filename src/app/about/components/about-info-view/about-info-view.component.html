About this application

The creation of this application started for fun when two new and passionate Topeleven players thought about ways to improve tracking and monitoring of players stats

Step by step, more ideas popped up and the functionality grew. Two months and a lot of free time spent later, the application offers various options not only related to players, but also the whole team, training and tactics.

Everyone who's interested is able to create an account and start right away! You'll find more information about data handling below.

A big THANK YOU to Iceman for offering so much free content on Discord and Reddit. All data regarding recommended training sessions and counter formations were taken from Discord.

What happens with my data?

Nothing!

The whole identity management is handled via Auth0 , a well-known and trusted identiy provider. To create an account, only an email-address and a nickname are needed. All other possible fields are optional. No data is stored in the applications database, except the auto- generated and unique Auth0 user id to identify your players and team.

Furthermore, this email will not used in any way! Since the developer itself is very privacy aware too, he's really not interested in spamming your inbox :).

All data is very protected by the Auth0 system, and it will not be used in any way other than provide an identity for this application.

Also, there are no trackers on this website whatsoever. It isn't even optimized for search engines, meanining that it's really just a plain and simple application for the sake of fun!
